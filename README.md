# Space Survival

## Реализовано на данный момент
1) В игре присутствует главное меню, меню выбора уровней и единственная на данный момент игровая сцена.
2) Меню выбора уровней содержит несколько кнопок, но все они отправляют на один уровень (для проверки системы разблокировки уровней)
3) Главное меню реализовано через UI Toolkit в целях ознакомления с технологией, остальной интерфейс сделан на привычном UI через Canvas
4) Игровая сцена подразумевает наличие игрока, спавнера противников и WinConditionChecker'а, который проверяет условие победы (пока что терпимо сделано только условие уничтожения всех противников)
5) Также WinCondition не будет работать при запуске сцены напрямую, потому что не будет инициализирован синглтон Levels для управления сценами
6) Игрок может перемещаться на WASD, целиться мышью и стрелять на ЛКМ
7) Противники появляются волнами с задержками (появление новой волны не зависит от уничтожения предыдущей)
8) ИИ противников элементарный - движутся на игрока и атакуют
9) Присутствует два вида оружия - плазменная пушка (обычная стрельба снарядами) и лазерный излучатель
10) Можно использовать несколько единиц вооружения одновременно
11) Присутствует механика выпадения и сбора валюты, а также подразумевается ее хранение, но пока не использование
12) Присутствует один вид противников
